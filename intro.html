LET'S START WITH C :

INTRODUCTION :

C is a general-purpose programming language created by Dennis Ritchie at the Bell Laboratories in 1972.
It is a very popular language, despite being old.
C is strongly associated with UNIX, as it was developed to write the UNIX operating system.
It is very easy to learn it includes all basics to begin as coder.
You will enjoy it !

Why learn C ?

Good question if you got in your mind!
Here are some reasons:

    

  • It is most popular programming language in the world.
    • 

  • If you know C, you will no problem learning other popular programming
languages such as Java, Python, C++, C#, etc, as the syntax is similar.
    • 

  • C is fast as compare to other programming languages.
    • 

  • C can be used in both applications and technologies.
    •
